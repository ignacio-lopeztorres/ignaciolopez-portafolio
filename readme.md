## How to Run Locally
General steps for cloning and running any project:

```bash
# clone repo
git clone https://github.com/username/project.git
cd project

# install
npm install    # or yarn

# run
npm start      # or npm run dev

# build
npm run build
```

Include environment variables:
```
# .env
DATABASE_URL=...
API_KEY=...
```
